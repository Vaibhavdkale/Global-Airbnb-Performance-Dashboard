# 🌍 Global Airbnb Performance Dashboard

## 📌 Project Overview

The **Global Airbnb Performance Dashboard** is an interactive Business Intelligence solution developed using **Power BI** to analyze Airbnb listings, pricing, customer reviews, host performance, market share, and seasonal booking trends across major global cities.

The dashboard transforms raw Airbnb data into meaningful business insights through interactive visualizations and KPI-driven analysis.

---

# 🎯 Business Problem

Airbnb operates across thousands of cities with millions of listings. Understanding customer behaviour, pricing trends, host performance, and seasonal demand is essential for improving business performance and customer experience.

This dashboard answers key business questions such as:

- Which cities have the highest Airbnb market share?
- Which property types generate the highest average price?
- Which cities receive the highest customer ratings?
- How do customer reviews change over time?
- What factors improve host trust?

---

# 🎯 Project Objectives

- Analyze Airbnb listings across major cities.
- Compare pricing across property types.
- Evaluate customer ratings and review trends.
- Study seasonal booking behaviour.
- Analyze host trust indicators.
- Generate business recommendations using data visualization.

# 📈 Key Insights

- Paris has the highest Airbnb market share with approximately 65K listings.
- Hotel Rooms have the highest average nightly price.
- Mexico City records the highest customer ratings.
- Nearly 86% of reviewers submitted only one review.
- Booking demand peaks between April and August.
- Most successful hosts have verified identities and profile pictures.

---

# 💡 Business Recommendations

- Expand listings in high-demand cities.
- Optimize pricing based on seasonality and property type.
- Increase Superhost participation.
- Strengthen host verification to improve customer trust.
- Launch targeted marketing campaigns during peak seasons.
